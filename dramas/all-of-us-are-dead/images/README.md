# All of Us Are Dead — Episode Images

Place episode thumbnail images in this folder.

## Naming convention
ep1.jpg, ep2.jpg ... ep12.jpg

## Recommended size
400 × 225 px (16:9) — JPEG or WebP, under 100 KB each

## How it works
- If ep1.jpg exists → shown as the episode card thumbnail
- If missing → emoji placeholder shown automatically
