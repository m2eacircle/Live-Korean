# Guardian — Episode Images

Place episode thumbnail images in this folder.

## Naming convention
ep1.jpg, ep2.jpg, ep3.jpg ... ep16.jpg

## Recommended size
400 × 225 px (16:9) or 300 × 200 px — JPEG or WebP, under 100 KB each

## How it works
- If ep1.jpg exists → shown as the thumbnail on the episode card and episode page header
- If the image is missing → the app automatically falls back to the emoji placeholder
- No code changes needed — just drop the correctly named file in this folder
